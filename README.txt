Сайт для приюта животных

Описание проекта
Сайт для приюта животных создан для упрощения взаимодействия с потенциальными усыновителями животных и для поддержки работы приюта. Сайт включает следующие основные разделы:

Новости: список актуальных событий и обновлений.
Животные: каталог животных в приюте с описанием, возрастом, полом и фотографиями.
Форум: платформа для обсуждения тем, связанных с уходом за животными, а также для обратной связи с пользователями.
Структура проекта
Проект реализован с использованием Django, фреймворка на языке Python.

Основные файлы и папки:
models.py: описание моделей базы данных (животные, новости, форум).
views.py: реализация логики отображения страниц и работы с данными.
urls.py: маршруты URL для доступа к различным страницам сайта.
templates/main: шаблоны для отображения веб-страниц.
static/main: статические файлы, включая CSS и изображения.
admin.py: настройки административной панели.

## Установка
1. Установите Python версии 3.12.
2. Создайте виртуальное окружение, если нет:
python -m venv venv
3. Активируйте окружение:
- Windows:
  .\venv\Scripts\activate
- Linux/MacOS:
  source venv/bin/activate
4. Установите зависимости:
pip install -r requirements.txt
5. Запустите сервер:
cd .\kind_soul\
python manage.py runserver

##Модели данных
1. News
Модель для хранения новостей.
Поля:
title — заголовок новости.
description — описание.
date — дата публикации.
image_url — URL изображения.

2. Animal
Модель для описания животных в приюте.
Поля:
name — имя.
age — возраст.
gender — пол (выбор между "Мальчик" и "Девочка").
description — описание.

3. AnimalPhoto
Модель для фотографий животных.
Поля:
animal — ссылка на животное.
photo_url — URL фотографии.

4. Topic (форум)
Модель для тем форума.
Поля:
title — название темы.
description — описание.
created_at — дата создания.
author — автор темы.

5. Comment (форум)
Модель для комментариев.
Поля:
topic — ссылка на тему.
content — текст комментария.
created_at — дата создания.
author — автор комментария.
